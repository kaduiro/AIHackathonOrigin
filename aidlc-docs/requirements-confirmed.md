# 要件確認完了

## 技術スタック

| レイヤー | 技術 |
|---|---|
| フロントエンド | Next.js (App Router) |
| UIライブラリ | shadcn/ui + Tailwind CSS |
| バックエンド | Next.js API Routes + Server Actions |
| データベース | Supabase (PostgreSQL) |
| 認証 | Supabase Auth |
| ストレージ | Supabase Storage |
| AI機能 | MVP段階ではモック実装（固定レスポンス / ルールベース） |

## プロダクト概要
学生向け「挑戦・学び・相談」コミュニティプラットフォーム MVP

## MVPスコープ（PRD準拠）
1. 会員登録 / ログイン
2. 年齢確認 / 保護者同意
3. 興味タグ選択 / 簡易診断
4. おすすめコミュニティ表示
5. タグフィード閲覧
6. 公開 Q&A 投稿 / コメント / リアクション
7. ルーム参加 / ルーム専用プロフィール
8. イベント作成 / 参加
9. 認証済みメンターへの予約制相談
10. AI チャットサポート（モック）
11. AI モデレーション（ルールベース）
12. AI プロフィール生成（モック）
13. 管理者機能（通報管理、審査、承認、停止、NGワード設定）

## 実装優先順位
1. 基盤: 認証 / 年齢確認 / 権限制御
2. オンボーディング / 興味タグ / 診断
3. ホーム / フィード / おすすめ
4. Q&A
5. ルーム
6. イベント
7. 認証済みメンター予約
8. AIプロフィール / AIチャット（モック）
9. AIモデレーション（ルールベース）
10. 管理者画面

## 安全設計の絶対ルール
- 非監督の私的DMを禁止
- 相談は監査可能な予約制導線上でのみ許可
- 全ユーザーに年齢確認必須
- 未成年は保護者同意必須
- メンター / コミュニティ作成者 / イベント主催者には追加審査
- 外部SNSのID交換・URL交換を禁止
- 違反投稿はAI（ルールベース）と人間運営の両方で判定

## 非機能要件
- モバイルファースト
- 監査ログ保持
- 個人情報の保護
- アクセス制御
- 基本的なアクセシビリティ
